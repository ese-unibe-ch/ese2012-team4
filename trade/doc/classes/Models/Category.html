Class Models::Category
In: app/models/module/category.rb
Parent: Object

Represents a category of different offers in the system. A category contains several subcategories. A category can return offers belonging to this category This class is not constructed to go deeper than three levels, that means:

  • First level: The constructed supercategory, that you can receive by invoking the get_supercategory method
  • Second level: Subcategories of the supercategory
  • Third level: Subcategories of the above mentioned subcategories of the supercategory
