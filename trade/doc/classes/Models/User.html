Class Models::User
In: app/models/module/user.rb
Parent: Object

Users have a name. Users have an amount of credits. A new user has originally 100 credit. A user can add a new item to the system with a name and a price; the item is originally inactive. A user provides a method that lists his/her active items to sell. User possesses certain items A user can buy active items of another user.
